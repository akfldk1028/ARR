# Workflow Architectures (LangGraph)

## 🎯 무엇을 하는 프로젝트인가요?

다양한 워크플로우 아키텍처 패턴을 학습하는 프로젝트입니다. LangGraph의 여러 디자인 패턴을 실습합니다.

## 📚 학습 내용

### 워크플로우 패턴:
- **Sequential**: 순차적 실행
- **Parallel**: 병렬 처리
- **Conditional**: 조건부 분기
- **Loop**: 반복 실행
- **Human-in-the-loop**: 사람 개입
- **Supervisor**: 감독자 패턴
- **Hierarchical**: 계층적 구조

## 📋 기술 스택

- **LangGraph**: 워크플로우 프레임워크
- **OpenAI**: 언어 모델
- **State Machine**: 상태 관리

## 🚀 실행 방법

### 1. 환경 변수 설정
`.env` 파일:
```env
OPENAI_API_KEY=your_api_key_here
```

### 2. Jupyter Notebook으로 학습
```bash
uv run jupyter notebook
```

각 노트북에서 다양한 패턴을 실습할 수 있습니다.
