# News Reader Agent (CrewAI)

## 🎯 무엇을 하는 프로젝트인가요?

뉴스를 수집하고 요약하는 AI 시스템입니다. 사용자 관심사에 맞는 뉴스를 찾아 간결하게 요약합니다.

## 🤖 CrewAI 에이전트 팀

- **News Collector**: 뉴스 수집
- **News Analyzer**: 뉴스 분석 및 분류
- **News Summarizer**: 요약 생성
- **Trend Detector**: 트렌드 분석

## 📋 기술 스택

- **CrewAI**: 에이전트 협업
- **Python 3.11**: ChromaDB 호환성

## 🚀 실행 방법

### 1. 환경 변수 설정
`.env` 파일:
```env
OPENAI_API_KEY=your_api_key_here
```

### 2. 실행
```bash
uv run python main.py
```

## 💡 기능

- 📰 실시간 뉴스 수집
- 🎯 관심사 기반 필터링
- 📝 요약 생성
- 📈 트렌드 분석

## 🔑 필요한 API 키

- ✅ OpenAI API Key (필수)
