# Multi-Agent Architectures (LangGraph)

## 🎯 무엇을 하는 프로젝트인가요?

다중 에이전트 시스템의 다양한 아키텍처 패턴을 학습하는 프로젝트입니다.

## 📚 학습 내용

### 아키텍처 패턴:
- **Swarm**: 분산 협업
- **Supervisor**: 관리자-작업자 패턴
- **Hierarchical**: 계층적 구조
- **Network**: 그래프 네트워크
- **Marketplace**: 에이전트 마켓플레이스

## 📋 기술 스택

- **LangGraph Supervisor**: 에이전트 조율
- **LangGraph Swarm**: 분산 시스템
- **OpenAI**: 언어 모델

## 🚀 실행 방법

### 1. 환경 변수 설정
`.env` 파일:
```env
OPENAI_API_KEY=your_api_key_here
```

### 2. 실행
```bash
uv run python graph.py
```

## 💡 배울 수 있는 것

- 다중 에이전트 설계 패턴
- 에이전트 간 통신
- 작업 분배 및 조율
- 복잡한 시스템 구축
