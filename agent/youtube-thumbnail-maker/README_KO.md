# YouTube Thumbnail Maker (LangGraph)

## 🎯 무엇을 하는 프로젝트인가요?

YouTube 썸네일을 자동으로 생성하는 AI 시스템입니다. 클릭률을 높이는 매력적인 썸네일을 만듭니다.

## 🤖 에이전트 기능

- 썸네일 콘셉트 아이디어 생성
- 텍스트 오버레이 최적화
- 색상 조합 추천
- A/B 테스트 버전 생성

## 📋 기술 스택

- **LangGraph**: 워크플로우 관리
- **OpenAI**: 콘셉트 생성
- **Image Generation**: 썸네일 이미지

## 🚀 실행 방법

### 1. 환경 변수 설정
`.env` 파일:
```env
OPENAI_API_KEY=your_api_key_here
```

### 2. 실행
```bash
uv run python graph.py
```

## 💡 기능

- 🎨 자동 디자인 생성
- 📝 텍스트 최적화
- 🔢 A/B 테스트 버전
- 🎯 클릭률 최적화

## 🔑 필요한 API 키
