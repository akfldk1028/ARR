# My First Agent

## 🎯 무엇을 하는 프로젝트인가요?

가장 기본적인 OpenAI 에이전트입니다. AI 에이전트의 기초를 배우기 위한 프로젝트입니다.

## 🤖 에이전트 설명

- **이름**: My First Agent
- **기능**: 기본적인 AI 대화 및 작업 수행
- **프레임워크**: OpenAI Python SDK
- **LLM 모델**: OpenAI GPT 모델

## 📋 기술 스택

- **OpenAI SDK**: 기본 에이전트 기능
- **Python**: 3.13

## 🚀 실행 방법

### 1. 환경 변수 설정
`.env` 파일에 OpenAI API 키를 입력하세요:
```env
OPENAI_API_KEY=your_api_key_here
```

### 2. Jupyter Notebook 실행
```bash
uv run jupyter notebook main.ipynb
```

또는 Python 스크립트로 실행할 수 있습니다.
