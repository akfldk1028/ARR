# src

Shared Python utilities for graph DB, LLM integration, and entity extraction.

## Key Files
- `main.py` - Graph creation and management utilities
- `graphDB_dataAccess.py` - Neo4j data access layer
- LLM integration helpers
- Entity extraction and chunk processing

## Note
NOT a Django app. General-purpose utility package.
Used by `law/` and `graph_db/` for data processing.
