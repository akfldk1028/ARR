# Radiance / honeybee-radiance

## Radiance
- 광선 추적(ray tracing) 기반 건물 광 시뮬레이션 표준
- LBNL 개발, 1989+
- Windows 설치: 별도 인스톨러 또는 Docker

## honeybee-radiance
- Python wrapper, Ladybug Tools
- UDI / sDA 자동 계산
- https://www.ladybug.tools/honeybee-radiance/

## pyradiance
- 더 lightweight Python wrapper
- Phase 1 A2 후보

## UDI 정의 (Reinhart 2014)
Useful Daylight Illuminance — 연중 100~2000 lux 받는 시간 비율

## sDA 정의 (LM-83)
spatial Daylight Autonomy — 300 lux 이상 자연광 받는 면적 비율 (50% 시간 기준)
